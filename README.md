# provi-osx

## My Mac OSX Provisioning
### How to Use
1. Git clone
```
mkdir $HOME/.repos
git clone --recursive https://github.com/yutarody/provi-osx.git  $HOME/.repos
```
2. Setup zprezto
```
zsh
zsh $HOME/.repos/provi-osx/setup_zprezto.zsh
```

3. Run script
```
sh $HOME/.repos/provi-osx/provi-osx.sh
```
